# C# WPF Path Finding Visualization

This is a c# WPF visualisation of the following path finding methods:
- A*
- Dijkstra
- Greedy


## Opening
You can use a editor or go to "bin" -> "Debug" -> "net10.0-windows" -> "PathFInding methods.exe"

## Editing

### - Start and end:
Double click in "Change end and start" click on the green ball (start) or the red ball (end) and move it to where you want then to be, after that click again to select the position.

### - Editing walls:
Double click in "Edit Walls: none", then left click to start adding walls and right click to start deleting walls.

### - Swiching algs:
You can swich the algorithm in the "Algorithms" tab.

### - Start visualisation:
Double click "GO".
